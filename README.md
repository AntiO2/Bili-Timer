# BiliTimer

## 功能说明

- 统计视频观看时间
  - 支持单个视频、分p视频、视频合集
  - 支持[番剧/影视](https://www.bilibili.com/bangumi)播放页
  - 支持[课堂](https://www.bilibili.com/cheese)播放页
- 支持百分比显示
- 动态图表。
- ![image](./assets/preview.jpeg)

## 使用方式

- 使用TamperMonkey即可安装
- 将鼠标移动到右侧查看进度信息。

## 使用到的工具

- TamperMonkey
- 原生JS
- Jquery
- Echarts
