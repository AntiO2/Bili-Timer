# BiliTimer

## 功能说明

- 统计视频观看时间
  - 支持单个视频、分p视频、视频合集
  - 暂时不支持[课堂](https://www.bilibili.com/cheese)功能
- 支持百分比显示
- 动态图表。
- ![image](https://tvax2.sinaimg.cn/large/0073fhLzly1h6lh2pn953j30jr0lcagm.jpg)

## 使用方式

- 使用TamperMonkey即可安装
- 将鼠标移动到右侧查看进度信息。

## 使用到的工具

- TamperMonkey
- 原生JS
- Jquery
- Echarts
